# 数据源侧边栏展示上次运行成功时间及定时刷新配置

数据源侧边栏需展示上次运行成功时间，并支持灵活的自动刷新配置。

## UI 需求
- 数据源侧边栏展示上次运行成功时间。
- 格式：使用 d/h/m 展示，仅展示最大的单位（例如 1d, 5h, 30m）。
- UI 样式：时间图标 + 时间字符串。

## 后端/逻辑需求
- 确定定时刷新的默认时间。
- 选项：关闭 / 5min / 30min / 1h 等。
- 配置层级：
  1. Setting 中设置全局刷新时间。
  2. 数据源条目下可指定特定数据源的刷新时间（优先级高于全局设置）。

## 验收标准
- [ ] 侧边栏能正确计算并显示上次成功运行的时间间隔。
- [ ] 全局设置中可配置默认刷新频率。
- [ ] 单个数据源可覆盖全局刷新频率。
- [ ] 自动刷新逻辑按配置执行。
